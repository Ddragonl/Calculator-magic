# 🎩 魔法计算器 (Magic Calculator)

一个神奇的计算器应用，能够通过三个特殊数字的相加，神奇地显示当前系统时间！

## ✨ 功能特点

- 🎯 **简单易用** - 界面简洁，操作流畅
- 🎨 **精美设计** - 仿手机系统计算器界面
- 📱 **跨平台** - 支持Web和Android APK
- ⚡ **离线运行** - 完全离线，无需网络连接

## 🎮 使用方法

1. **输入第一个数字**：输入你最开心的日期
   - 格式：`YYYYMMDD`（年月日，8位数字）
   - 例如：2025年2月5日 → 输入 `20250205`

2. **点击加号 (+)**，然后输入第二个数字：输入你最伤心的日期
   - 格式：`YYYYMMDD`（年月日，8位数字）
   - 例如：2025年11月3日 → 输入 `20251103`

3. **再次点击加号 (+)**，然后输入第三个数字：随机输入任意数字
   - 可以输入任意长度的数字
   - 建议闭上眼睛随机输入，增加神秘感！

4. **点击等号 (=)**：查看神奇的结果！

## 🎪 神奇效果

当你完成三个数字的输入并按下等号后，计算器会显示一个10位数字，这个数字正好对应**当前系统时间**！

**时间格式说明：**
- 格式：`YYMMDDHHmm`（年月日时分，10位数字）
- 例如：2026年1月1日 20:13 → 显示 `2601012013`

## 📱 安装方式

### Web版本
直接在浏览器中打开 `index.html` 即可使用。

### Android APK
1. 下载 `app-debug.apk` 文件
2. 在Android手机上允许"安装未知来源应用"
3. 点击APK文件进行安装
4. 安装完成后即可使用
